---

Most of what's here started as a weekend experiment. I wanted to *actually understand* an algorithm or a CSS trick or a game mechanic, so I built something around it.

The stack is always the same — TypeScript, React, Vite — because I stopped fighting my toolchain and started shipping instead. 53 projects in, still learning things from each one.

---

## What I'm working on

Been going deeper into pathfinding lately — **algo-vis** has A*, Dijkstra, BFS, DFS and I want to add maze generation algorithms next (recursive backtracking is interesting). Also slowly cleaning up **task-manager** now that the Drizzle ORM integration is in better shape.

The **portfolio** site pulls everything live from the GitHub API, which means it stays current automatically. That was the goal.
